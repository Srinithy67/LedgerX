# LedgerX

LedgerX is a modern expense management platform that combines manual expense tracking with automated transaction capture.

Users can upload payment screenshots, review extracted transactions, and organize expenses through a streamlined workflow.

## Features

### Expense Tracking

- Manual expense entry
- Transaction history
- Payment method tracking
- Category management

### Smart Capture

- Screenshot uploads
- OCR-based transaction extraction
- Review before saving
- Automated transaction detection

### Analytics

- Monthly spending reports
- Category breakdowns
- Expense trends
- Spending summaries

### Personalization

- Matcha Strawberry theme
- future themes are planned keeping in mind a ocean blue and a lilac theme

## Architecture

User

→ Supabase Database

Screenshot Upload
→ OCR Processing
→ Transaction Review
→ Expense Storage
